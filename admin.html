Normalisation & Keys

PRIMARY KEYS
orders.id (UUID), products.id (SERIAL), customers.id (SERIAL), coupons.id (SERIAL)
FOREIGN KEYS
orders.customer_id → customers.id · order_items.order_id → orders.id · order_items.product_id → products.id
INDEXES
orders: created_at, payment_status, fulfillment · products: category, active · customers: email, phone
3NF NORMALISATION
order_items table separates product data from orders · categories table · tiers stored as JSONB
